# history/v001/proposed_changes

The proposed-changes queue captured in the `v001` snapshot. `v001` is
the initial seed revision, so no proposed changes were accepted into it
— this queue is empty. The directory is retained because every
`history/vNNN/` snapshot carries its `proposed_changes/` subdir (per
`livespec/SPECIFICATION/spec.md` §"`version-directories-complete` …").
